git init 创建管理仓库
git add . 添加所有文件到暂存区
git commit 添加暂存区到仓库
git status  查看是否有文件未添加（红）提交（绿）
git diff 文件名 查看文件修改了什么内容
git log 显示从最近到最远的提交日志 git log –pretty=oneline（省略部分内容显示）(实际使用过程中没信息展示)
cat +文件名 展示当前文件具体内容
git reset --hard HEAD^返回到上一个版本 git reset --hard HEAD~100 返回到前100个版本 git reset --hard 版本号 回到目标版本
git reflog 获取到所有的版本号
